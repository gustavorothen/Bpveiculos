# BP Veículos – Catálogo Digital

Este repositório contém o site desenvolvido para a **BP Veículos**, uma loja de automóveis que necessitava de uma solução digital simples, moderna e acessível. O projeto foi criado utilizando **HTML, CSS/SCSS, JavaScript e Bootstrap**, com foco em responsividade, organização e facilidade de manutenção.

## Tecnologias utilizadas

- **HTML5** – Estrutura das páginas  
- **CSS3 / SCSS** – Estilização, responsividade e organização modular  
- **Bootstrap 5** – Componentes, grid e layout moderno  
- **JavaScript** – Interatividade, eventos e recursos dinâmicos  
- **Git & GitHub** – Versionamento e publicação do código  

---

## Funcionalidades Principais

- Layout totalmente responsivo para desktop e mobile  
- Navbar dinâmica carregada via `partials/header.html`  
- Rodapé modular via `partials/footer.html`  
- Cards de veículos na página inicial  
- Página de estoque com listagem visual  
- Página sobre com informações da loja  
- Página de contato com mapa integrado  
- Botão flutuante “voltar ao topo” usando JavaScript  
- Estrutura de código limpa e de fácil manutenção
